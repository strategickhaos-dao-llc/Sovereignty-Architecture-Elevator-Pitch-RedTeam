## üéì UNDERSTANDING PROVISIONAL PATENTS

### What is a Provisional Patent?

A **provisional patent application** is:
- ‚úÖ A USPTO filing that establishes a priority date
- ‚úÖ Less formal than a utility patent (no claims required, though we include them)
- ‚úÖ Not examined by USPTO (no approval/rejection)
- ‚úÖ Confidential (not published)
- ‚úÖ Valid for 12 months only
- ‚úÖ Must be converted to non-provisional to get actual patent

### What You Can Do With a Provisional

**After filing:**
- ‚úÖ Use "Patent Pending" marking
- ‚úÖ Publicly disclose invention without losing rights
- ‚úÖ Discuss with investors/partners
- ‚úÖ Market product with IP protection notice
- ‚úÖ Begin commercial activities

**You cannot:**
- ‚ùå Enforce patent rights (no patent issued yet)
- ‚ùå Sue infringers (no enforceable patent)
- ‚ùå License the patent (just the pending application)
- ‚ùå Extend beyond 12 months (absolute deadline)

### Why File a Provisional?

**Advantages:**
1. **Lower cost:** $75 vs $600+ for non-provisional
2. **Less formal:** Can file quickly without perfect claims
3. **Priority date:** Establishes "first to file" timestamp
4. **Flexibility:** 12 months to refine invention before committing to full patent
5. **Market testing:** Can commercialize while deciding if full patent is worthwhile

**Use case for Sovereignty Architecture:**
- Technology is novel and working
- Architecture is documented thoroughly
- Want to secure priority date now
- Will spend next 12 months validating market demand
- Can decide later if full patent prosecution is worthwhile

## üåç INTERNATIONAL PROTECTION

### Options for Global Coverage

If you want patent protection outside the US:

**Option 1: Patent Cooperation Treaty (PCT)**
- File PCT application within 12 months of provisional
- Single application covers 150+ countries
- Costs ~$4,000 to file
- 30 months to decide which countries to pursue
- **Recommended for global protection**

**Option 2: Direct Foreign Filing**
- File separately in each desired country
- Must file within 12 months of provisional
- Different fees per country
- More complex but sometimes cheaper for few countries

**Option 3: US-Only**
- Only file US non-provisional
- No international protection
- Lowest cost (~$600-$6,000 total)
- **Sufficient if targeting US market only**

### Cost Comparison (Estimated)

- **US only:** $6,000 lifetime (DIY, micro entity)
- **US + PCT (5 countries):** $30,000-$50,000
- **US + Europe + Asia (comprehensive):** $100,000+

**Strategy for Sovereignty Architecture:**
- Start with US provisional (today)
- Assess market traction over 12 months
- File PCT if showing global market potential
- Focus on US if primarily domestic opportunity

### Alternative to Patent

If you decide **not** to pursue a full patent, consider **defensive publication**:

**What it is:**
- Publicly disclosing the invention in detail
- Prevents others from patenting the same idea
- Keeps technology in public domain
- No patent rights for you, but also no rights for competitors

**When to use:**
- Patent prosecution costs are prohibitive
- Technology moves too fast for patent lifecycle
- Open source strategy is preferred
- Want to prevent competitor patents without expense

**For Sovereignty Architecture:**
- GitHub repository already serves as defensive publication
- This patent documentation could be published if not filed
- Consider if open source community building > patent protection

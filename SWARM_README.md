## ğŸ“œ Governance Guidelines

The Sovereign Swarm operates on principles of **distributed authority** and **emergent order**:

### Decision Making

**For Code Changes**:
- Small changes: Open PR, get 1 review from any contributor, merge

We don't have "owners"â€”we have **stewards**:

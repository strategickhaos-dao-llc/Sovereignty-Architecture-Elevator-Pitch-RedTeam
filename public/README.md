# Patent Center UI

This directory contains the Patent Center web application UI.

## Overview

The Patent Center is a web-based application for managing patent-related workflows. This implementation is built using Angular and includes:

- USPTO components integration
- Qualtrics feedback widget
- Session timeout management
- Bootstrap-based responsive design
- Public Sans and Roboto Mono web fonts

## Key Features

- **Modern Web Fonts**: Uses Public Sans and Roboto Mono for optimal readability
- **Feedback System**: Integrated Qualtrics website feedback mechanism
- **Session Management**: Built-in session timeout widget
- **Responsive Design**: Bootstrap-based responsive layout
- **Angular Framework**: Built with Angular for dynamic component rendering

## Monitoring History

**Note**: As of April 22, 2025, Instana monitoring has been retired (April 24, 2025). The monitoring integration has been removed from this application.

## Structure

- `index.html` - Main application entry point
- `assets/` - Static assets (stylesheets, session timeout widget)
- `thirdParty/` - Third-party font files (Roboto Mono, Public Sans)
- `*.js` - Application JavaScript modules and polyfills

## Integration

This UI component is designed to integrate with the broader Sovereignty Architecture ecosystem, providing a specialized interface for patent management workflows within the sovereign infrastructure platform.

## Compilation Instructions

### Option 1: Overleaf (Recommended)

1. Go to [Overleaf.com](https://overleaf.com)
2. Create a new project → Upload Project
3. Upload the `.tex` file
4. Click "Recompile" to generate the PDF
5. Download the compiled PDF

**Note:** Overleaf will automatically handle the `fontspec` and `Libertinus Serif` font requirements.

### Option 2: Local Compilation with XeLaTeX

```bash
# Requires XeLaTeX for fontspec support
xelatex Garza_2025_Neurodivergent_Biomimetic_Swarm.tex
xelatex Garza_2025_Neurodivergent_Biomimetic_Swarm.tex  # Run twice for references
```

**Requirements:**
- XeLaTeX or LuaLaTeX (for `fontspec` package)
- Libertinus Serif font installed (or modify to use Computer Modern)
- Standard LaTeX packages: `geometry`, `hyperref`, `booktabs`, `xcolor`

### Option 3: pdfLaTeX (Alternative)

If you don't have XeLaTeX, modify the document by removing/commenting out these lines:

```latex
% \usepackage{fontspec}
% \setmainfont{Libertinus Serif}
```

Then compile with:

```bash
pdflatex Garza_2025_Neurodivergent_Biomimetic_Swarm.tex
pdflatex Garza_2025_Neurodivergent_Biomimetic_Swarm.tex  # Run twice
```

## Publication Venues

### 1. Google Scholar

**Steps:**
1. Register for ORCID at [orcid.org](https://orcid.org) (takes 60 seconds)
2. Log into Google Scholar with your SNHU email: `domenic.garza@snhu.edu`
3. Go to "My Profile" → "Add" → "Upload"
4. Upload the compiled PDF
5. The paper will appear under "Domenic Garza" profile
6. Update ORCID field in the document once registered

**Link your ORCID:** After registration, update the ORCID link in the LaTeX document from:
```
ORCID: \href{https://orcid.org/0000-0003-XXXX-XXXX}{pending registration}
```
to your actual ORCID identifier.

### 2. arXiv (Optional)

**Steps:**
1. Create account at [arxiv.org](https://arxiv.org)
2. Submit to category: `cs.AI` (Artificial Intelligence) or `cs.MA` (Multiagent Systems)
3. Upload the compiled PDF
4. Fill in metadata (title, abstract, authors)
5. arXiv will assign a permanent identifier (e.g., `arXiv:2511.XXXXX`)

**Note:** arXiv requires endorsement for first-time submitters in some categories.

### 3. Zenodo (Recommended for DOI)

**Steps:**
1. Create account at [zenodo.org](https://zenodo.org)
2. Click "Upload" → "New upload"
3. Upload the PDF
4. Select publication type: "Pre-print"
5. Fill in metadata (authors, affiliations, keywords)
6. Click "Publish"
7. Zenodo will assign a permanent DOI (e.g., `10.5281/zenodo.XXXXXXX`)

**Advantage:** Instant DOI assignment (5 minutes), permanent archival storage

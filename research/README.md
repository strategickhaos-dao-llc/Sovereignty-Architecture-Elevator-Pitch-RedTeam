## Integration Guide

### For Overleaf/LaTeX Papers

1. Copy `trading_engine_design.tex` into your project
2. Add `trading_engine_references.bib` entries to your bibliography
3. Include with `\input{trading_engine_design.tex}`
4. Compile: `pdflatex main.tex; bibtex main; pdflatex main.tex x2`

### For Obsidian/Google Docs

1. Use `TRADING_ENGINE_DESIGN.md` content directly
2. Copy-paste sections as needed
3. Update reference numbers to match your existing bibliography

### For arXiv Submission

1. Use LaTeX version for consistent formatting
2. Ensure all citations are properly formatted
3. Include supplementary materials as appendices

## üí° Pro Tips

### Pipe to Clipboard
```bash
# macOS
curl -L -s URL | pbcopy

# Linux with xclip
curl -L -s URL | xclip -selection clipboard

# Linux with xsel
curl -L -s URL | xsel --clipboard
```

### Direct to AI Chat
```bash
# Get content and feed to your local AI
curl -L -s URL | your-llm-cli "Summarize this"

# Or paste into Continue.dev, Claude, etc.
```

### Batch Download
```bash
# Download all at once (use the full script)
./curl_knowledge_library.sh

# Or create a custom batch
for url in $(cat urls.txt); do
  curl -L -s "$url" -o "$(basename $url)"
done
```

### Save with Metadata
```bash
# Add timestamp to filename
curl -L -s URL -o "content_$(date +%Y%m%d).txt"

# Save with source URL in filename
curl -L -s URL -o "$(echo URL | md5sum | cut -d' ' -f1).txt"
```

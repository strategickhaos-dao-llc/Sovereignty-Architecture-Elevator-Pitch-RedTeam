## Pre-Execution Checklist

### Property Identification
- [ ] All IP assets precisely identified (copyrights, patents, code, etc.)
- [ ] Revenue streams clearly defined
- [ ] Future derivatives and renewals addressed
- [ ] No conflicting prior assignments or encumbrances
- [ ] Ownership verified and documented

### Beneficiary Verification
- [ ] All beneficiaries are qualified 501(c)(3) organizations
- [ ] EINs verified for each beneficiary
- [ ] Current addresses and contacts confirmed
- [ ] Proportional shares determined (must sum to 100% of 7%)

### Legal Instrument Selection
- [ ] Direct Irrevocable Assignment selected (recommended)
- [ ] OR Charitable Lead Trust (CLUT) if trust wrapper needed
- [ ] Governing law specified (Wyoming recommended)
- [ ] Attorney review scheduled/completed

## Execution Checklist

### Document Preparation
- [ ] Assignment document drafted with all required elements:
  - [ ] Irrevocability clause
  - [ ] Present assignment of future royalties
  - [ ] Governing law
  - [ ] Split schedule
  - [ ] Audit rights
  - [ ] Notice addresses
  - [ ] Exhibit A (IP enumeration)
  - [ ] Exhibit B (contact/payment details)

### Signing and Notarization
- [ ] Document signed by Assignor
- [ ] Document notarized
- [ ] Notary commission verified current
- [ ] Notary seal and acknowledgment complete

### Delivery (Completes the Gift)
- [ ] Executed copies delivered to each charity
- [ ] Delivery method documented (certified mail, email, in-person)
- [ ] Delivery receipts obtained
- [ ] Written acknowledgments requested

## IRS Substantiation Checklist

### For ANY Contribution ≥ $250
- [ ] Written acknowledgment obtained from each charity
- [ ] Acknowledgment includes:
  - [ ] Description of contributed property
  - [ ] Statement whether goods/services provided in exchange
  - [ ] Description and FMV of any goods/services provided
- [ ] Acknowledgment obtained by earlier of:
  - [ ] Date tax return filed, OR
  - [ ] Due date (including extensions) for return

### For Noncash Contributions > $500
- [ ] Form 8283 required
- [ ] Section A completed (for contributions $501-$5,000)
- [ ] Records maintained:
  - [ ] Name and address of each charity
  - [ ] Description of contributed property
  - [ ] Date of contribution
  - [ ] How property was acquired
  - [ ] Donor's cost or adjusted basis
  - [ ] FMV at time of contribution and method of determination

### For Noncash Contributions > $5,000
- [ ] Qualified appraisal obtained
- [ ] Appraisal completed no earlier than 60 days before contribution
- [ ] Appraisal completed no later than due date of return (with extensions)
- [ ] Form 8283 Section B completed
- [ ] Appraiser qualifications verified:
  - [ ] Relevant professional education and experience
  - [ ] Meets IRS qualified appraiser requirements
  - [ ] Appraiser declaration signed (Part III of Section B)
- [ ] Donee acknowledgment obtained (Part IV of Section B)
- [ ] Appraisal summary attached to return

### For Intellectual Property Contributions
- [ ] Aware that additional deductions may be available if donated IP produces income for charity
- [ ] Charity informed of Form 8899 reporting requirements
- [ ] Tracking system established for charity income reports

## Trust Route (If Applicable - CLUT/CRUT)

### Additional Requirements for Trust Approach
- [ ] Formal trust document drafted meeting:
  - [ ] 26 U.S.C. §170(f)(2)(B) requirements
  - [ ] 26 U.S.C. §664 regulations (for lead/remainder variants)

## Red Flags to Avoid

### Common Mistakes
- [ ] NOT filing trust with Wyoming SOS thinking it makes it irrevocable (FALSE)
- [ ] NOT calling it a CRUT when it doesn't meet §664 requirements
- [ ] NOT taking deduction without proper substantiation
- [ ] NOT missing the acknowledgment deadline
- [ ] NOT providing false valuations on Form 8283

### Audit Risk Factors
- [ ] Contribution value over $500,000 (high audit probability)
- [ ] No qualified appraisal for property > $5,000
- [ ] Missing donee acknowledgment
- [ ] Inconsistent valuation methodology
- [ ] Related-party transactions

For Statement of Use filing, you'll need "specimens" showing actual use:

### For Goods (Class 9 - Software)
- Screenshots showing mark on software interface
- Product packaging/download pages
- User manuals with mark
- Software documentation with mark

### For Services (Classes 35, 36, 41, 42)
- Advertisements showing mark with services
- Website pages describing services
- Business cards/letterhead
- Marketing materials
- Promotional screenshots

**Specimen Format:**
- JPG or PDF format
- Clear, readable mark
- Shows mark used in commerce
- Dated if possible

## âš ï¸ EVIDENCE BEST PRACTICES

1. **Timestamp Everything**
   - Include dates in filenames
   - Preserve metadata
   - Use Wayback Machine for web pages

2. **Multiple Copies**
   - Store in GitHub repo
   - Backup to cloud storage
   - Keep physical copies of key documents

3. **Chain of Custody**
   - Document who collected evidence
   - Note collection date and method
   - Maintain audit trail

4. **Quality Standards**
   - High-resolution screenshots
   - Legible documents
   - Complete pages (not cropped)

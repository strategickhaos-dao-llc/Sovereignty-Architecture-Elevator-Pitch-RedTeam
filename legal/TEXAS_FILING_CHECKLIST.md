## Document Preparation

### Address Usage Rules
1. **Registered Agent Section**: Use ONLY Registered Agents Inc address
2. **Principal Office Section**: Use ONLY Longview business address
3. **Mailing Address**: Use Longview business address (NOT registered agent)

### Common Mistakes to Avoid
- ❌ Using registered agent address for principal office
- ❌ Using registered agent address for federal filings

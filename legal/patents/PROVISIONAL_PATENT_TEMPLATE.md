## TIPS FOR COMPLETING THIS TEMPLATE

### Writing the Description

**Be Detailed:** 
- More detail is always better than less
- Explain not just WHAT but WHY and HOW
- Include technical specifics where relevant
- Describe alternatives and variations

**Use Clear Language:**
- Technical but understandable
- Define specialized terms
- Use consistent terminology throughout
- Number or label components consistently

**Reference Drawings:**
- Explicitly refer to figures in text
- Example: "As shown in Figure 1..."
- Label all parts in drawings
- Explain what each drawing shows

### Creating Drawings

**Quality Requirements:**
- Clear and legible
- Labeled components
- Multiple views if needed
- Reference numbers match text

**Types of Drawings:**
- System architecture diagrams
- Flowcharts for methods
- Block diagrams
- Screenshots (for software/UI)
- Hardware schematics
- Network topology
- Data flow diagrams

## AFTER COMPLETING THIS TEMPLATE

### Review Your Work

1. Read through completely
2. Check that all sections are filled out
3. Ensure drawings match descriptions
4. Verify technical accuracy
5. Look for gaps or unclear sections
6. Have someone else review if possible

### Convert to PDF

1. Save this document as PDF
2. Save all drawings/figures as PDF
3. Combine into single PDF if desired (or upload separately)

### File Online

1. Go to: https://www.uspto.gov/patents/apply
2. Log into EFS-Web system
3. Select "Provisional Application"
4. Upload your PDF documents
5. Complete micro-entity certification
6. Pay $65 filing fee
7. Submit application
8. Save confirmation and filing receipt

### What Happens Next

- USPTO will assign application number
- You receive filing date (your priority date)
- You can use "Patent Pending" 
- You have 12 months to file non-provisional (if desired)
- Your application remains confidential

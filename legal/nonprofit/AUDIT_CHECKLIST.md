## Appendix: Audit Automation

### Automated Checks
The following checks can be automated using the audit script:

```bash
# Run automated audit checks
./audit_nonprofit.sh \
  --month "YYYY-MM" \
  --verify-hashes \
  --verify-signatures \
  --check-arweave \
  --check-receipts \
  --financial-reconciliation \
  --output-report "audit_YYYY-MM.pdf"
```

### Manual Review Required
The following items require human judgment and manual review:
- Board meeting quality and completeness
- Alignment of activities with mission
- Appropriateness of expenses
- Conflict of interest evaluations
- Strategic recommendations

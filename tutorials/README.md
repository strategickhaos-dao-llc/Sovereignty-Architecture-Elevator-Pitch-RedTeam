### [Data Visualization with Python](data-visualization/)

Learn Python data visualization fundamentals using matplotlib, pandas, and numpy.

**Topics Covered:**
- Python basics: data types and data structures
- Module imports and common data science libraries
- Creating plots with matplotlib
- Line charts and overlay bar charts
- Real-world examples with unemployment and traffic data

**Quick Start:**
```bash
# Install dependencies
pip install -r requirements.visualization.txt

# Run an example
python tutorials/data-visualization/unemployment_plot.py

# Or use Jupyter notebooks
jupyter notebook tutorials/data-visualization/
```

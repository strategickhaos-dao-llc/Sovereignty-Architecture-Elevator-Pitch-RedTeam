## Quick Start

### Bulk Download (One-Line)

```bash
# Download all 100 papers in one command
cd arsenal/papers
xargs -n 1 curl -L -s -O < ../arsenal.txt
```

### Alternative: Using wget

```bash
cd arsenal/papers
wget -i ../arsenal.txt
```

### Download with Progress

```bash
cd arsenal/papers
cat ../arsenal.txt | grep -v "^#" | xargs -I {} bash -c 'echo "Downloading: {}"; curl -L -s -O "{}"'
```

## Integration with RAG Systems

### For LangChain

```python
from langchain.document_loaders import DirectoryLoader, PDFLoader

# Load all papers
loader = DirectoryLoader('./arsenal/papers/', glob="**/*.pdf", loader_cls=PDFLoader)
documents = loader.load()

# Create embeddings and vector store
from langchain.embeddings import OpenAIEmbeddings
from langchain.vectorstores import Chroma

embeddings = OpenAIEmbeddings()
vectorstore = Chroma.from_documents(documents, embeddings)
```

### For LlamaIndex

```python
from llama_index import SimpleDirectoryReader, VectorStoreIndex

# Load papers
documents = SimpleDirectoryReader('./arsenal/papers/').load_data()

# Create index
index = VectorStoreIndex.from_documents(documents)
```

### For Custom RAG

Use the provided `arsenal_metadata.jsonl` file for embedding and retrieval:

```bash
# Each line contains: {"url": "...", "title": "...", "category": "...", "id": N}
cat arsenal/arsenal_metadata.jsonl | jq -r '.url'
```

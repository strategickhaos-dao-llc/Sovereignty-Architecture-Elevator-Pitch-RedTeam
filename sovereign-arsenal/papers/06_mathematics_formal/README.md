# Category 06: Mathematics & Formal Methods

This category contains 10 papers on mathematical foundations, type theory, and formal verification.

## Papers Included

1. **Category Theory for Programmers** - Bartosz Milewski
   - Category theory applied to programming
   - Compositional systems

2. **Homotopy Type Theory** - Various authors
   - Foundations of mathematics
   - Constructive proof theory

3. **Higher Category Theory** - Various authors
   - Advanced categorical structures
   - N-categories and infinity-categories

4. **Dependent Types and Program Equivalence** - Various authors
   - Types that depend on values
   - Expressive type systems

5. **Formal Verification of Smart Contracts** - Various authors
   - Proving contract correctness
   - Preventing vulnerabilities

6. **Formal Methods in Blockchain** - Various authors
   - Mathematical verification of protocols
   - Provable security

7. **Simply Typed Lambda Calculus** - Various authors
   - Foundational type system
   - Basis of functional programming

8. **Proof Assistants: History and Applications** - Various authors
   - Coq, Isabelle, Lean
   - Interactive theorem proving

9. **Computational Logic and Verification** - Various authors
   - Logic for program verification
   - Model checking

10. **Foundations of Probabilistic Programming** - Various authors
    - Formal semantics for probabilistic code
    - Bayesian inference

## Why These Papers Matter

Formal methods enable provably correct systems:
- **Type theory** prevents entire classes of bugs
- **Category theory** enables compositional design
- **Formal verification** proves correctness
- **Proof assistants** make proofs practical

Essential for:
- Critical systems
- Smart contracts
- Security-critical code
- Provably correct software

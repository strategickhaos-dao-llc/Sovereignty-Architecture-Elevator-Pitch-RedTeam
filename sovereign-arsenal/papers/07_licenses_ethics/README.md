# Category 07: Open Source Licenses & Ethics

This category contains 10 foundational open source licenses and ethical frameworks.

## Licenses Included

1. **GNU General Public License v3 (GPL-3.0)**
   - Strong copyleft license
   - Ensures software freedom

2. **MIT License**
   - Permissive open source
   - Maximum freedom

3. **Apache License 2.0**
   - Permissive with patent grant
   - Corporate-friendly

4. **Mozilla Public License 2.0 (MPL-2.0)**
   - File-level copyleft
   - Balanced approach

5. **Creative Commons BY-SA 4.0**
   - Share-alike for content
   - Attribution required

6. **GNU Affero General Public License (AGPL-3.0)**
   - Network copyleft
   - Closes SaaS loophole

7. **Hippocratic License**
   - Ethics-first licensing
   - Human rights protection

8. **Ethical Source Licenses**
   - Family of ethical licenses
   - Values-based distribution

9. **Free Software Definition** - FSF
   - Four essential freedoms
   - Philosophy of freedom

10. **Open Source Definition** - OSI
    - 10 criteria for open source
    - Community standard

## Why These Licenses Matter

Licenses are code. Code is law. Choose wisely:
- **GPL/AGPL** protect software freedom with copyleft
- **MIT/Apache** enable maximum permissiveness
- **MPL** balances freedom and business use
- **Ethical licenses** encode values into code distribution
- **Definitions** provide philosophical grounding

Essential for:
- Choosing project licenses
- Understanding software freedom
- Building sovereign systems
- Protecting digital commons

## License Comparison

| License | Type | Copyleft | Patent Grant | Use Case |
|---------|------|----------|--------------|----------|
| GPL-3.0 | Strong Copyleft | Yes | Implicit | Software freedom |
| AGPL-3.0 | Network Copyleft | Yes | Implicit | SaaS applications |
| MIT | Permissive | No | No | Maximum flexibility |
| Apache-2.0 | Permissive | No | Explicit | Corporate use |
| MPL-2.0 | Weak Copyleft | File-level | Explicit | Hybrid projects |

## Recommended Reading Order

1. **Free Software Definition** for philosophy

3. **MIT** and **GPL** for the spectrum endpoints
4. **Apache 2.0** for patent considerations
5. **AGPL** for network services

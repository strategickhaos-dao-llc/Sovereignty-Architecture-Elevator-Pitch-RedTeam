# Heir Icon

Place your red heart icon here as `heir.png` for the Continue.dev sidebar.

Recommended specifications:
- Format: PNG with transparency
- Size: 32x32 or 64x64 pixels
- Color: Red/pink theme (#ff0044)
- Style: Heart icon or custom heir symbol

Example sources for heart icons:
- https://www.flaticon.com/free-icons/heart
- https://iconmonstr.com/favorite-3-png/
- Create your own in GIMP, Inkscape, or Figma

Once you have your icon, save it as `heir.png` in this directory.

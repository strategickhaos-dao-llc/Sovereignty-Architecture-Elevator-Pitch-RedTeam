# Brokerage Selection Checklist for Nonprofit Organizations

**SNHU Ecosystem Fund**  
**Prepared:** November 2025

---

## Purpose

This checklist helps the Treasurer and Investment Committee evaluate and select an appropriate brokerage firm for managing the Organization's investment portfolio in accordance with the Investment Policy Statement.

---

## Top Three Recommended Brokerages for Nonprofits

### 1. Fidelity Investments
### 2. Charles Schwab
### 3. Interactive Brokers (IBKR)

---

## Evaluation Criteria

Use this checklist when evaluating each brokerage option:

### ✅ Account Setup & Eligibility

- [ ] **Nonprofit Account Support**: Firm offers dedicated nonprofit/institutional accounts
- [ ] **EIN Documentation**: Can open account using Organization's EIN
- [ ] **Minimum Balance**: Initial minimum is achievable (typically $0-$25,000)
- [ ] **501(c)(3) Status**: Accepts applications from pending or approved 501(c)(3) organizations
- [ ] **Online Application**: Streamlined digital application process available

### ✅ Fees & Costs

- [ ] **No Annual Fee**: No or low annual account maintenance fees
- [ ] **Commission-Free ETFs**: Access to commission-free ETF trading
- [ ] **No Transaction Fees**: Low or zero fees for mutual fund purchases
- [ ] **Management Fees**: Competitive advisory fees if using managed account services (typically 0.25%-0.85%)
- [ ] **Hidden Fees**: No excessive inactivity, transfer, or account closing fees

### ✅ Investment Options

- [ ] **Broad ETF Selection**: Access to VTI, VXUS, BND, ESGU, and similar funds
- [ ] **ESG Investments**: Strong selection of ESG-screened funds and ETFs
- [ ] **Fixed Income**: Comprehensive bond and fixed-income options
- [ ] **Money Market**: FDIC-insured or highly-rated money market funds
- [ ] **Fractional Shares**: Ability to purchase fractional shares (helpful for smaller portfolios)

### ✅ Platform & Tools

- [ ] **User-Friendly Interface**: Intuitive web and mobile platforms
- [ ] **Research Tools**: Access to investment research and screening tools
- [ ] **Portfolio Analysis**: Built-in performance tracking and reporting
- [ ] **Automatic Rebalancing**: Option for automated portfolio rebalancing
- [ ] **Tax Reporting**: Comprehensive tax documents and reporting (even if tax-exempt)

### ✅ Governance & Compliance

- [ ] **IPS Upload**: Ability to submit and link IPS to account
- [ ] **Multiple Signers**: Support for multiple authorized signers
- [ ] **Board Oversight**: Regular statements suitable for Board review
- [ ] **Audit Support**: Provides documentation needed for annual audits
- [ ] **UPMIFA Compliance**: Familiar with nonprofit fiduciary standards

### ✅ Customer Service

- [ ] **Nonprofit Specialist**: Dedicated support for institutional/nonprofit accounts
- [ ] **Phone Support**: Direct phone line to account representatives
- [ ] **Educational Resources**: Webinars, guides, and tools for nonprofit investing
- [ ] **Response Time**: Reasonable response times for inquiries (24-48 hours)

### ✅ Integration & Advanced Features

- [ ] **API Access**: Available for portfolio tracking and analysis (if needed)
- [ ] **Third-Party Integration**: Compatible with portfolio management software
- [ ] **Custodial Services**: Can serve as custodian for external RIA if needed
- [ ] **Futures Access**: Can link to platforms like NinjaTrader for analysis/hedging (if applicable)

## Decision Process

### Step 1: Initial Research
- [ ] Visit each brokerage's nonprofit account page
- [ ] Request information packet for institutional accounts
- [ ] Review fee schedules and account minimums

### Step 2: Application Preparation
- [ ] Gather required documents (EIN letter, bylaws, board resolution, IPS)
- [ ] Identify authorized signers from board
- [ ] Prepare initial funding method

### Step 3: Account Opening
- [ ] Complete online application or schedule call with nonprofit specialist
- [ ] Submit required documentation
- [ ] Set up online access and security features

### Step 4: Initial Setup
- [ ] Link funding source (bank account)
- [ ] Upload Investment Policy Statement
- [ ] Configure transaction authorities and limits
- [ ] Set up automatic statements and reporting

### Step 5: First Investments
- [ ] Transfer initial funds
- [ ] Execute first purchases according to IPS asset allocation
- [ ] Verify transactions and confirmations
- [ ] Document all activity for Board report

## Recommended Choice for Small-to-Medium Nonprofits

### **Fidelity Investments or Charles Schwab**

**Rationale:**
- Both offer excellent nonprofit account support with dedicated specialists
- Extensive commission-free ETF selection including ESG options
- User-friendly platforms suitable for volunteer Board members
- Strong educational resources and customer service
- Well-established compliance and audit support
- $0 minimum balance requirements
- Straightforward online application process

**Interactive Brokers** is recommended only if:
- You have sophisticated investment needs
- Multiple currency or international investment requirements exist
- You prefer professional-grade tools and are comfortable with complexity
- You plan to use advanced hedging strategies or futures

---

## Next Steps After Selection

1. **Schedule Consultation**: Book call with nonprofit account specialist
2. **Prepare Documents**: Gather IPS, board resolution, EIN letter, bylaws
3. **Open Account**: Complete application with all required documentation
4. **Initial Funding**: Transfer seed capital from operating account
5. **Implement IPS**: Execute initial investments per asset allocation targets
6. **Board Report**: Present account opening and first investment summary to Board

# Governance Documents

This directory contains governance documents for the SNHU Ecosystem Fund and related organizational structures.

## Investment Management Documents

These documents provide a complete framework for managing the organization's investment portfolio in accordance with nonprofit best practices and the Uniform Prudent Management of Institutional Funds Act (UPMIFA).

### Core Documents

1. **[Investment Policy Statement](investment_policy_statement.md)** (`investment_policy_statement.md`)
   - Comprehensive guidelines for prudent investment management
   - Asset allocation targets and ranges
   - ESG screening criteria
   - Risk management and liquidity requirements
   - **Status**: Ready for Board adoption

2. **[Board Resolution for IPS Adoption](board_resolution_ips_adoption.md)** (`board_resolution_ips_adoption.md`)
   - Formal board resolution template for adopting the IPS
   - Establishes Investment Committee authority
   - Delegates responsibilities to Treasurer
   - **Usage**: Complete and sign at board meeting when adopting IPS

3. **[Brokerage Selection Checklist](brokerage_selection_checklist.md)** (`brokerage_selection_checklist.md`)
   - Comprehensive evaluation criteria for selecting a brokerage firm
   - Comparison of Fidelity, Charles Schwab, and Interactive Brokers
   - Step-by-step decision process
   - Contact information for nonprofit account specialists
   - **Usage**: Use when opening investment accounts

4. **[Quarterly Investment Report Template](quarterly_investment_report_template.md)** (`quarterly_investment_report_template.md`)
   - Complete template for quarterly board investment reports
   - Performance metrics and benchmarking
   - Asset allocation tracking
   - Transaction details and fee analysis
   - Compliance certification
   - **Usage**: Complete quarterly for Investment Committee/Board review

## Implementation Workflow

### Step 1: Board Adoption (Week 1)
- [ ] Review Investment Policy Statement with full Board
- [ ] Vote on adoption using Board Resolution template
- [ ] Sign and date both IPS and Resolution
- [ ] File in organizational records

### Step 2: Brokerage Selection (Week 2-3)
- [ ] Complete Brokerage Selection Checklist
- [ ] Contact 2-3 preferred brokerages for information
- [ ] Compare fees, features, and nonprofit support
- [ ] Make selection and prepare account opening documents

### Step 3: Account Opening (Week 3-4)
- [ ] Gather required documents (EIN letter, bylaws, signed IPS, board resolution)
- [ ] Complete brokerage account application
- [ ] Set up authorized signers and online access
- [ ] Link funding source

### Step 4: Initial Investment (Month 2)
- [ ] Transfer initial capital to brokerage account
- [ ] Execute purchases according to IPS asset allocation
- [ ] Document all transactions
- [ ] Verify trades and confirmations

### Step 5: Ongoing Management (Quarterly)
- [ ] Prepare Quarterly Investment Report using template
- [ ] Review performance against benchmarks
- [ ] Check asset allocation vs. IPS targets
- [ ] Rebalance if necessary
- [ ] Present report to Board

### Step 6: Annual Review (Yearly)
- [ ] Conduct comprehensive IPS review
- [ ] Update asset allocation targets if needed
- [ ] Review ESG screening criteria
- [ ] Assess brokerage relationship
- [ ] Update spending policy if necessary

## Usage Guidelines

### Adding New Trademark Evidence

1. Create folder under `trademarks/[MARK_NAME]/`
2. Add first-use evidence with timestamps
3. Include screenshots, archives, and commit logs
4. Generate SHA256 hashes of all evidence
5. Update the provenance log

### Adding New Invention Disclosure

1. Copy `templates/invention_disclosure_template.yaml`
2. Create folder under `patents/[INVENTION_ID]/`
3. Complete all sections of the disclosure
4. Attach supporting documentation
5. Submit for DIPI prior art research

### Documentation Standards

- **Timestamps**: Always use ISO 8601 format (YYYY-MM-DDTHH:MM:SSZ)
- **Hashes**: Generate SHA256 for all evidence files
- **Naming**: Use format `[TYPE]_[NAME]_[DATE]_[VERSION].[ext]`
- **Metadata**: Include creation date, creator, and classification

## Important Notes

1. **First-Use Evidence**: Critical for establishing priority. Document EVERYTHING.
2. **Public Disclosure**: Starts 1-year clock for US patents, may bar foreign rights.
3. **Confidentiality**: Mark sensitive documents appropriately.
4. **Retention**: All IP provenance records retained permanently.

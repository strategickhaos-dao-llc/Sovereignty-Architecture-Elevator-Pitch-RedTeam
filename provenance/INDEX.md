# Provenance Index

This directory contains cryptographic provenance records for all releases.

## Overview

The Sovereignty Architecture uses a multi-layer provenance system:

1. **BLAKE3 Hashing** - Fast, secure cryptographic hashes for all bundles
2. **GPG Signatures** - Detached signatures for verification
3. **IPFS Pinning** - Content-addressed storage for permanent availability
4. **Cosign Signing** - Keyless signing via Sigstore for supply chain security
5. **SBOM Generation** - Software Bill of Materials via Syft

## Verification

To verify a bundle:

```bash
# Using the verify CLI tool
./bin/verify swarmgate_v1.0.tar.gz

# Manual verification
b3sum swarmgate_v1.0.tar.gz
gpg --verify swarmgate_v1.0.tar.gz.asc swarmgate_v1.0.tar.gz
cosign verify-blob --bundle swarmgate_v1.0.tar.gz.sig swarmgate_v1.0.tar.gz
```
